# next-docs-starter
starter project for nextjs docs

## Clone the repo
```
git clone https://github.com/brendenvogt/next-docs-starter
cd next-docs-starter
```

## Run the dev server 
```
yarn
yarn dev
```

## Generate/export static website 
```
yarn build
```

## Deploy
```
yarn plan
yarn apply
```

## Destroy
```
yarn destroy
```
